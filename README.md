# RBAC School Backend

This backend codebase is being developed to understand the **design and concepts of a Role-Based Access Control (RBAC) system**.

## Business Context

The application is modeled around a **school system** to simulate real-world RBAC scenarios such as:

- Super Admin access
- School Admin management
- Teacher-level permissions
- Section-based data access

## Backend Tech Stack

- **TypeScript**
- **Node.js (Express)**
- **MySQL**
- **Prisma ORM**
- **JWT (Authentication & Authorization)**

## Purpose

This project focuses on:

- Understanding RBAC architecture
- Implementing permission-based access control
- Designing scalable backend systems
- Learning middleware-based authorization
